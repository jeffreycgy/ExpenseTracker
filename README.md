# ExpenseTracker

An expense tracking app built with React Native with Redux &amp; Saga

> Note: This app is tested XCode iOS Simulator

## Setup

1. Clone the repo

```
git clone https://github.com/jeffreycgy/ExpenseTracker.git
```

2. Checkout to project folder

```
cd ExpenseTracker
```

3. Install packages

```
yarn
cd ios && pod install
```

4. Run app

```
yarn ios
```
